# Event & life photos — just drop images here

Any image you put in this folder (`.jpg`, `.png`, `.webp`) automatically appears
on the **/gallery/** page ("Life at Trishakti") the next time the site builds.
No code needed.

## How to add a photo
- **From your computer:** copy the image file into this folder.
- **From GitHub (phone or laptop, no tools):** open this folder on github.com →
  "Add file" → "Upload files" → drag your photos in → "Commit changes".
  The site rebuilds automatically in about a minute.

## Tips
- **Name the file to caption it.** `dashain-team-lunch-2025.jpg` shows as
  "Dashain Team Lunch 2025". Generic names like `IMG_1234.jpg` show no caption.
- **Order them** by starting names with numbers: `01-`, `02-`, `03-` …
- Photos of the team, the floor, festivals, training, and buyer visits work best.
- Don't upload photos of anyone who hasn't agreed to appear on the website.

The gallery link only appears on the site once at least one photo is here.
