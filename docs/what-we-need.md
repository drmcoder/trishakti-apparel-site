# What we need from Trishakti Apparel to build the export site

Check items off as you provide them. Anything not provided will be **omitted** (no invented specs).

## 1. Sourcing facts (the numbers buyers decide on)
- [ ] Monthly production capacity — real pieces/month (e.g. "X pcs/month across N lines")
- [ ] MOQ — minimum order quantity per style, and per color/size
- [ ] Lead time — sampling time + bulk production time (in days/weeks)
- [ ] Incoterms handled — EXW / FOB (which port? Kathmandu/Kolkata?) / CIF
- [ ] Sample policy — free / paid, how long, who pays courier
- [ ] Payment terms accepted — T/T, L/C, deposit %, etc.
- [ ] Certifications / audits — any held now (OEKO-TEX, GOTS, BSCI, Sedex, WRAP) or "in progress / none yet"
- [ ] Export status — EXIM code (yes), REX for EU (in progress), any others

## 2. Product line card (what you make)
For each product type (t-shirt, polo, hoodie, etc.):
- [ ] Product name / category
- [ ] Fabric types you run (e.g. single jersey, pique, fleece) + GSM ranges
- [ ] Size range offered
- [ ] Per-style MOQ (if different)
- [ ] 1–3 photos per product

## 3. Photos & assets (most important — this sells credibility)
- [ ] Logo — SVG or high-res PNG (transparent background)
- [ ] Factory-floor photos — sewing line, cutting room, QC, packing (real, well-lit)
- [ ] Product photos — t-shirts/polos on hangers or flat-lay, close-up stitch detail
- [ ] Team / leadership photo(s)
- [ ] Building exterior / signage shot
- [ ] Any brand color / font guide if one exists

## 4. Positioning decisions
- [ ] Primary tagline for the hero (buyer-focused, e.g. "Private-label knitwear
      manufacturer — your designs, made in Nepal"). Keep the poetic one as secondary?
- [ ] Which markets to name explicitly (EU / US / UK / "worldwide")
- [ ] Client logos you're allowed to display (or keep testimonials text-only)
- [ ] Languages — English only, or add others?

## 5. Contact & inquiry
- [x] Address, phones, WhatsApp, email — already have (see build-prompt.md)
- [ ] Which email should inquiries land in? (admin@trishaktiapparel.com or another)
- [ ] Anyone specific to name as the export/sales contact?
